О проекте:
	- Данное решение написано в программе Visual Studio 2022
	- Данный решение состоит из 3 проектов: Sibers.MVC, Sibers.Core, Sibers.Services
	- Данный проект написан на на платформе .NET 9.0

О запуске проекта:
	1) Если вы получили архив с приложением и читаете данный абзац значит вы скорее всего уже разархивировали архим, если же нет то первым действие разархивируйте архив.
	2) Убедитесь что на вашем компьютере установлен пакет SDK версии .NET 9.0 (скачать можно по данной ссылке https://dotnet.microsoft.com/ru-ru/download)
	   Как только вы скачали или же если у вас был установлен пакет откройте командную строку (Win + R, cmd) и введите команду dotnet --version, проверьте версию вашего пакета
	   Если все правильно идете к следующему шагу
	3) Откройте файл решения с помощью IDE Visual studio 2022, Rider и т.д. 
	4) В командой строке в папке с проектов нужно ввести команду dotnet restore для загрузки всех необходимых зависимостей и библиотек
	5) Выберите в качестве запускамеого проекта Sibers.MVC и запустите.

Об архитектуре проекта:
	- Данный проект следует трехуровневой архитектуре что значит что в нем есть уровень доступа к данных, уровень логики и уровень пользовательского интерфейса
	- Данный решение состоит из 3 проектов: 
		1) Sibers.MVC(Presentation Layer, пользовательский интерфейс), 
		2) Sibers.Services(Business Logic Layer, бизнес-логика)
		3) Sibers.Core(Data Access Layer, доступ к базе данных)
	- Данный проект использует в качестве базы данных базу Sqlite
